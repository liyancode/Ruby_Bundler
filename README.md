## Bundler 是什么？
> Bundler能够跟踪并安装所需的特定版本的gem，以此来为Ruby项目提供一致的运行环境。  
> Bundler是Ruby依赖管理的一根救命稻草，它可以保证你所要依赖的gem如你所愿地出现在开发、测试和生产环境中。利用Bundler启动项目简单到只用一条命令：** bundle install **  

## 上手
Bundler上手非常简单，第一步打开命令行输入如下命令：  
'''ruby
gem install bundler
'''
